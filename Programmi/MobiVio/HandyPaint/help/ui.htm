User Interface

HandyPaint provides flexible, easy to use user interface for you.

1. Toolbar

You can resize the toolbar by clicking and dragging bottom/bottom-right edge of toolbar to rearrange buttons in it. You also can move the toolbar by clicking and dragging the left stick on it.


2. Canvas

The area that you can draw in.


3. Zoom Bar

This bar gives you options for image stretching. You can click the "Zoom in/Zoom out" button on the Toolbar to show/hide this bar.


4. Thumbnail Window

This window displays a thumbnail of your painting. You can click the "Pan" button on the toolbar to show this window.


5. Pencil Hardness Panel

You can expand or collapse this panel. You can use the slider in this panel to change the pencil's hardnesss.


6. Command Panel

You can expand or collapse this panel. There are several commands contained in this panel, such as Create New Paint, Save Paint, Save Paint as Image File, Send, and Open Paint.


7. Color Window

This window displays the current using color for drawing. You can click on it for selecting a new color.
